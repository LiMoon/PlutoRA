『import』資料夾的用處為何？
-------------------------------------------------------------------------------

該資料夾提供讓你可以保存自己的設定，而不用煩惱每次更新是否會覆蓋到既有的檔案。
你可以儲存自己的設定，而剩餘的設定就與 rAthena 保持同步。

如何運行？
-------------------------------------------------------------------------------

僅需把你要改變的設定放進 import 資料夾即可。
舉例來說，若你想更改 /battle/exp.conf 的設定：

	// 基礎經驗值倍率 (Note 2)
	base_exp_rate: 700

你只需要把這一條設定指令複製貼到 /import/battle_conf.txt 就可以囉！
而剩下的設定就會維持原本的樣子，未來也不用擔心更新會造成不便。

簡單又方便，對吧？
